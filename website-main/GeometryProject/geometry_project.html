About This Project

This project was created during my freshman year geometry class to demonstrate a real-world application of trigonometry. It features a gun mechanic where bullets are spawned using trigonometric calculations.

Features

  • Real-world application of trigonometry
  • Bullet spawning mechanics based on angles
  • Interactive and visually engaging

Technologies Used

  • Godot Engine
  • GD Script
  • Aseprite
