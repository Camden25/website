About This Game

Made as a personal project

Game Overview

Circular pong + Flappy Bird. You have to keep the ball up while avoiding obstacles like in Flappy Bird but only using a paddle that spins around a circle like in circular pong.

Features

  • Interesting Gameplay
  • Randomly Generated Levels
  • Power Ups
  • Crisp 2D Vector Art
